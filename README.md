## Perfect Pay

### Instruções para rodar o projeto:

- Clonar o projeto
- Rodar composer install
- Criar .env a partir do .env.example
- Criar banco de dados local
- Rodar migrations (php artisan migrate)
- Instanciar servidor (php artisan serve)
